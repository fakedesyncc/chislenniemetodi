# Теория численных методов

## 1. Численное интегрирование

Численное интегрирование - это процесс приближенного вычисления определенного интеграла функции. Оно применяется, когда аналитическое решение интеграла невозможно или слишком сложно.

### 1.1 Метод трапеций

Метод трапеций основан на аппроксимации подынтегральной функции линейной функцией. Интервал интегрирования [a, b] разбивается на n равных частей, и на каждом отрезке функция аппроксимируется трапецией.

Формула метода трапеций:

∫(a,b) f(x)dx ≈ (b-a)/n * [(f(a) + f(b))/2 + ∑(i=1 to n-1) f(a + i*h)]

где h = (b-a)/n - шаг интегрирования.

Погрешность метода трапеций имеет порядок O(h²).

### 1.2 Метод Симпсона

Метод Симпсона (или метод парабол) аппроксимирует подынтегральную функцию квадратичной функцией. Интервал интегрирования [a, b] разбивается на четное число n равных частей, и на каждой паре соседних отрезков функция аппроксимируется параболой.

Формула метода Симпсона:

∫(a,b) f(x)dx ≈ (b-a)/(3n) * [f(a) + f(b) + 4*∑(i=1,3,5... to n-1) f(a + i*h) + 2*∑(i=2,4,6... to n-2) f(a + i*h)]

где h = (b-a)/n - шаг интегрирования.

Погрешность метода Симпсона имеет порядок O(h⁴).

### 1.3 Метод Ньютона-Котеса

Метод Ньютона-Котеса основан на интерполяции подынтегральной функции полиномом Лагранжа. В данной реализации используется формула Ньютона-Котеса четвертого порядка (3/8 правило).

Формула метода Ньютона-Котеса (3/8 правило):

∫(a,b) f(x)dx ≈ (b-a)/8 * [f(a) + 3*f(a + h) + 3*f(a + 2h) + f(b)]

где h = (b-a)/3.

Погрешность метода Ньютона-Котеса имеет порядок O(h⁴).

### 1.4 Принцип Рунге

Принцип Рунге используется для оценки погрешности численного интегрирования. Он основан на вычислении интеграла с разным числом разбиений и сравнении результатов.

Если I₁ - значение интеграла при n разбиениях, а I₂ - значение интеграла при 2n разбиениях, то погрешность можно оценить по формуле:

ε ≈ |I₂ - I₁| / (2^p - 1)

где p - порядок точности метода (p = 2 для метода трапеций, p = 4 для метода Симпсона).

## 2. Интерполяция

Интерполяция - это метод нахождения промежуточных значений величины по имеющемуся дискретному набору известных значений.

### 2.1 Интерполяционный многочлен Лагранжа

Интерполяционный многочлен Лагранжа - это полином минимальной степени, проходящий через заданный набор точек.

Формула многочлена Лагранжа:

L(x) = ∑(i=0 to n) yi * ∏(j=0 to n, j≠i) (x - xj) / (xi - xj)

где (xi, yi) - заданные точки.

Преимущества:
- Простота формулы
- Легкость вычисления для небольшого числа точек

Недостатки:
- Сложность добавления новых точек (требуется полный пересчет)
- Возможны большие осцилляции при большом числе точек

### 2.2 Интерполяционный многочлен Ньютона

Интерполяционный многочлен Ньютона использует разделенные разности для построения полинома.

Формула многочлена Ньютона:

N(x) = f[x0] + f[x0,x1](x-x0) + f[x0,x1,x2](x-x0)(x-x1) + ... + f[x0,...,xn](x-x0)...(x-x(n-1))

где f[xi,...,xj] - разделенные разности.

Преимущества:
- Легкость добавления новых точек без полного пересчета
- Более устойчив к осцилляциям, чем многочлен Лагранжа

Недостатки:
- Более сложная формула и вычисление, чем у многочлена Лагранжа

### 2.3 Сравнение методов интерполяции

Оба метода (Лагранжа и Ньютона) дают одинаковый результат для одного и того же набора точек, так как они строят полином одной и той же степени. Основные различия заключаются в способе построения полинома и удобстве использования в различных ситуациях.

Выбор метода зависит от конкретной задачи:
- Для небольшого числа точек и одноразового вычисления удобнее использовать метод Лагранжа.
- Для большого числа точек или при необходимости добавления новых точек лучше использовать метод Ньютона.

### 2.4 Оценка погрешности интерполяции

Погрешность интерполяции зависит от гладкости функции и расстояния между узлами интерполяции. Для полинома степени n погрешность оценивается как O(h^(n+1)), где h - максимальное расстояние между соседними узлами.

Важно помнить, что увеличение степени интерполяционного полинома не всегда приводит к уменьшению погрешности, особенно на краях интервала (феномен Рунге).

## 3. Математические основы

### 3.1 Определенный интеграл

Определенный интеграл функции f(x) на отрезке [a, b] определяется как:

∫(a,b) f(x)dx = lim(n→∞) ∑(i=1 to n) f(ξᵢ)Δxᵢ

где Δxᵢ = (b-a)/n, а ξᵢ - некоторая точка из отрезка [xᵢ₋₁, xᵢ].

Геометрически определенный интеграл представляет собой площадь под графиком функции f(x) на отрезке [a, b].

### 3.2 Полиномиальная интерполяция

Задача интерполяции состоит в построении функции, проходящей через заданный набор точек. Полиномиальная интерполяция использует полином степени не выше n для интерполяции n+1 точек.

Теорема: Для любых n+1 различных точек (x₀, y₀), (x₁, y₁), ..., (xₙ, yₙ) существует единственный полином степени не выше n, проходящий через эти точки.

### 3.3 Разделенные разности

Разделенные разности - это коэффициенты в представлении интерполяционного полинома Ньютона. Они определяются рекурсивно:

f[xᵢ] = f(xᵢ)
f[xᵢ, xᵢ₊₁] = (f[xᵢ₊₁] - f[xᵢ]) / (xᵢ₊₁ - xᵢ)
f[xᵢ, xᵢ₊₁, ..., xᵢ₊ₖ] = (f[xᵢ₊₁, ..., xᵢ₊ₖ] - f[xᵢ, ..., xᵢ₊ₖ₋₁]) / (xᵢ₊ₖ - xᵢ)

Разделенные разности имеют важное свойство: f[x₀, x₁, ..., xₙ] = f^(n)(ξ) / n!, где ξ - некоторая точка из отрезка [min(xᵢ), max(xᵢ)].
