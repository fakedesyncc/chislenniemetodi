# Справка по использованию программы

## 1. Численное интегрирование

1. Перейдите на вкладку "Интегрирование".

2. Введите функцию f(x) в поле "Функция f(x)". Используйте стандартный синтаксис Python.
   Доступные функции: sin, cos, tan, exp, log (логарифм по основанию 10), ln (натуральный логарифм), sqrt, abs
   Примеры:
   - x**2 (квадрат x)
   - sin(x) (синус x)
   - exp(-x**2) (экспонента от -x²)
   - x**3 + 2*x**2 - 3*x + 1 (полином третьей степени)

3. Введите нижний и верхний пределы интегрирования в соответствующие поля.

4. Введите требуемую точность вычислений в поле "Точность (eps)".

5. Выберите метод интегрирования:
   - "Все методы" - будут использованы все три метода с последующим сравнением результатов
   - "Метод трапеций" - только метод трапеций
   - "Метод Симпсона" - только метод Симпсона
   - "Метод Ньютона-Котеса" - только метод Ньютона-Котеса

6. Нажмите кнопку "Вычислить".

7. Результаты вычислений и графики будут отображены в правой части окна.
   - График функции
   - Визуализация метода трапеций
   - Визуализация метода Симпсона
   - Сравнение результатов всех методов

8. В текстовом поле результатов вы увидите подробное описание вычислений для каждого метода,
   включая промежуточные шаги, итоговый результат, число разбиений и оценку погрешности.

## 2. Интерполяция

1. Перейдите на вкладку "Интерполяция".

2. Введите узловые точки в формате "x y" (по одной точке на строку) в текстовое поле.
   Пример:
   0 0
   1 1
   2 4
   3 9

3. Введите точку интерполяции x* в соответствующее поле.

4. Выберите метод интерполяции:
   - "Оба метода" - будут использованы оба метода с последующим сравнением результатов
   - "Многочлен Лагранжа" - только метод Лагранжа
   - "Многочлен Ньютона" - только метод Ньютона

5. Нажмите кнопку "Интерполировать".

6. Результаты интерполяции и графики будут отображены в правой части окна.
   - График исходных точек и интерполяционных многочленов
   - Визуализация погрешности интерполяции
   - График базисных функций Лагранжа
   - Визуализация таблицы разделенных разностей

7. В текстовом поле результатов вы увидите подробное описание вычислений для каждого метода,
   включая промежуточные шаги, итоговый результат и оценку погрешности.

## 3. Дополнительные функции

### 3.1 Переключение темы

Для переключения между светлой и темной темами используйте меню "Вид":
1. Выберите "Вид" в верхнем меню
2. Выберите "Светлая тема" или "Темная тема"

Светлая тема использует цветовую схему в стиле macOS Light, а темная тема - в стиле macOS Dark.

### 3.2 Сохранение результатов

Для сохранения результатов в PDF:
1. Выберите "Файл" в верхнем меню
2. Выберите "Сохранить результаты"
3. Укажите имя и расположение файла в появившемся диалоговом окне
4. Нажмите "Сохранить"

PDF-файл будет содержать все графики и текстовые результаты вычислений.

### 3.3 Теоретическая информация

Для получения теоретической информации о методах перейдите на вкладку "Теория".
Здесь вы найдете подробное описание всех используемых методов, их математические формулы и свойства.

### 3.4 Справка

Для получения справки по использованию программы перейдите на вкладку "Справка" (текущий раздел).

## 4. Особенности интерфейса

Программа имеет современный интерфейс в стиле macOS/iOS с поддержкой светлой и темной тем.

Основные элементы интерфейса:
- Вкладки для переключения между разделами
- Поля ввода для параметров вычислений
- Кнопки для запуска вычислений
- Графики для визуализации результатов
- Текстовые поля для отображения подробных результатов
- Строка состояния в нижней части окна

Все элементы интерфейса адаптируются к выбранной теме, обеспечивая комфортную работу в любых условиях освещения.

## 5. Системные требования

- Python 3.6 или выше
- Библиотеки: tkinter, numpy, matplotlib, sympy
- Рекомендуемое разрешение экрана: 1280x800 или выше

При возникновении вопросов или затруднений обратитесь к разделу "Теория" для получения 
дополнительной информации о методах численного интегрирования и интерполяции.
